# 단기 알바 전용 급여 & 스케줄 기록기

## 1. 한 문장 설명
장기 아르바이트와 달리 근무지와 시간이 매번 바뀌는 '단기 알바'에 특화된 급여 기록 프로그램입니다.

## 2. 풀려는 문제 및 동기
현재 시중의 알바 앱들은 한 곳에서 오래 일하는 경우에만 최적화되어 있어, 날짜마다 근무지가 바뀌는 단기 알바생이 쓰기에 불편합니다. 
- 여러 장소에서 일한 내역을 날짜별로 관리하고, 총 수입이 얼마인지 정확히 확인하기 어렵습니다.
- 제가 직접 겪는 이 불편함을 Python 코딩을 통해 해결해보고자 합니다.

## 3. 핵심 기능
- **필수** 날짜별 근무 장소, 시간, 시급 입력 및 저장 기능
- **필수** 이번 달 총 근무 횟수와 전체 예상 수입 합계 출력
- **있으면 좋음** 건당 교통비나 식대 등 추가 비용 포함 계산 기
  
## 4. 예상 사용자
- 여러 업체를 옮겨 다니며 단기 아르바이트를 하는 대학생 (본인 포함)

## 5. 결과물 형태
- CLI

## 6. 사용할 Python 개념
- **리스트와 딕셔너리:** 날짜별 알바 내역 저장
- **반복문(for):** 전체 내역을 확인하며 총 수입 합산
- **함수:** 데이터 입력 및 결과 출력 기능 분리
